# Arduino Learning Notes
## Date
## Topic
NeoPixel LED Rainbow Running Light
## Goal
Learn how to control LED ring and create a rainbow running effect in Arduino IDE
---
## Hardware
- Arduino Board
- NeoPixel LED ring
- VCC -> 5V
- GND -> GND
- IN  -> PWM Pin (for example D6)
---
## Example Code
```cpp
#include <Adafruit_NeoPixel.h>

#define PIN 6
#define NUMPIXELS 16

Adafruit_NeoPixel pixels(NUMPIXELS, PIN, NEO_GRB + NEO_KHZ800);

void setup() {
  pixels.begin();
}

void loop() {

  for(int i=0; i<NUMPIXELS; i++) {

    pixels.setPixelColor(i, pixels.Color(255,0,0));
    pixels.show();
    delay(100);

    pixels.setPixelColor(i, pixels.Color(0,0,0));
  }

}
Explanation

NUMPIXELS = number of LEDs on the ring

setPixelColor() sets the color of a specific LED

pixels.show() sends the data to the LEDs

delay() controls the speed of the running light

The loop lights one LED after another, creating a running effect.
