# Lernfeld 04: Einfache IT-Systeme bereitstellen (AWE)

DE
## Lernziele
- Einfache Services fuer Entwicklung bereitstellen
- Benutzer, Rechte und Zugriff sauber setzen
- Ergebnisse dokumentieren

## Muss koennen
- Lokale Entwicklungsumgebung betreiben
- Grundlegende Services starten (Webserver, DB lokal)
- Zugriff und Berechtigungen korrekt setzen

## AP1 Bezug (AWE)
- Praktische Basisaufgaben, saubere Dokumentation

## AP2 Bezug (AWE)
- Servicebereitstellung und Abnahme

## Uebungen
- Starte einen lokalen Webserver und dokumentiere Schritte
- Lege einen Benutzer an und setze Rechte
- Schreibe ein kurzes Abnahmeprotokoll

## Checkliste
- Service laeuft stabil
- Rechte korrekt gesetzt
- Dokumentation nachvollziehbar
